## Run Locally

Clone the project

```bash
  git clone https://github.com/azamat0v/qr_akademiya.git
```

Go to the project directory

```bash
  cd qr_akademiya
```

Install dependencies

```bash
  pub get
```

Start the server

```bash
  flutter run
```
